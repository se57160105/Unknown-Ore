GAME TITLE
----------
Unknown Ore

GENRE
-----
Roblox mining / collection / reveal simulator.

CORE FANTASY
------------
Mine something unknown.
Discover the container rarity.
Choose whether to fuse or refine it.
Analyze it over time.
Manually reveal the real specimen.
Collect, hold, display, sell, and optimize specialized Shelf builds.

CORE LOOP
---------
ENTER CAVE
→ MINE PERSONAL NODE
→ RECEIVE UNKNOWN CONTAINER
→ MINING BACKPACK
→ RETURN TO HOME BASE
→ CHOOSE AN ACTION (separate, optional actions - not a fixed order/sequence):
   - ANALYZE ELIGIBLE ITEMS (input source not yet locked; see 04_ANALYZE_DESIGN)
   - DEPOSIT ITEMS INTO LAB STORAGE FOR LATER USE (not an Analyze prerequisite)
   - OPTIONAL FUSION
   - OPTIONAL REFINERY
   - FREE BACKPACK SPACE THROUGH ANOTHER APPROVED INVENTORY ACTION
→ WAIT / FINISH NOW
→ MANUAL REVEAL
→ ACTUAL ITEM INSTANCE
→ HOLD / COLLECT / SHELF / SELL
→ BUY UPGRADES
→ UNLOCK NEXT CAVE

CORE PILLARS
------------
1. Unknown loot and reveal anticipation.
2. 96-item collection.
3. Physical Shelf build management.
4. Specialized Luck / Sell Price / Analyze Speed / Mining Speed builds.
5. Medium main progression and long-term optimization.
6. Social visibility without shared-node competition.

WORLD BASELINE
--------------
Max players per server: 10.
Each player owns one private 72x72-stud plot.
Ten plots are arranged around a central shared hub.
Target Homebase footprint: approximately 320x320 studs.
Target Plot → Cave Access travel time: no more than 10–15 seconds.

Private means ownership permission, not isolation.
Visitors may enter and inspect public Shelf/Trophy displays.
Visitors may not mutate another player's items, Shelf, Lab, Refinery, Sell state, or rewards.

HOMEBASE IDENTITY
-----------------
Compact Personal Mining Research Yard.

Primary areas:
- Ore Intake / Lab Storage
- Analyze Lab
- Fusion / Refinery Area
- Central Shelf / Mineral Display Area
- Ore Exchange / Sell Terminal
- Collection Trophy Area
