# Unknown Ore

Unknown Ore is a Roblox mining, analysis, collection, and display game.

This repository uses a Project OS: a small set of authoritative documents that allows ChatGPT, Claude, Codex, Cursor, or a human developer to restore project context before working.

## Start here

Read in this order:

1. `docs/project-os/00_START_HERE.md`
2. `docs/project-os/01_PROJECT_STATE.md`
3. `docs/project-os/02_AI_HANDBOOK.md`
4. `docs/tickets/active/ACTIVE_TICKET.md`
5. The design and architecture files relevant to the task

## Core rule

Project files are the source of truth. AI memory and chat history are not authoritative.

## Repository areas

- `docs/project-os` — project state, working rules, changelog, source-of-truth map
- `docs/design` — approved game design
- `docs/architecture` — runtime and data architecture
- `docs/roadmap` — ordered future work
- `docs/tickets` — current and completed implementation tickets
- `docs/reviews` — independent review reports
- `docs/decisions` — approved design and architecture decisions
- `templates` — reusable ticket, review, decision, and state-update templates
